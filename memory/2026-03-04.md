### 3. Valencia Construction Lead Nurture Sequence
**File:** `valencia-lead-nurture-sequence.md`
**What:** SMS + Facebook Messenger + Email funnel to convert cold leads → consultations → estimates → signed contracts
**Structure:**
- SMS automation (3 texts, same day, within 5-8 hours)
  - Immediate response (thanks for form, quick question)
  - 6h follow-up (pricing context, value-add)
  - 8h follow-up (customized based on response)
- Facebook Messenger (3 messages, days 1-3)
  - Credibility + transparency
  - Overcome objections
  - Direct pressure (if still no booking)
- Pre-call email
  - Reduce no-shows
  - Get context
- Post-call sequences
  - If YES → Estimate delivery + payment terms
  - If MAYBE → Objection handling + decision motivation
  - If NO → Soft follow-up + 30-day value add

**Timeline:** Form → Consultation (4-5 days), Consultation → Estimate (5-6 days), Estimate → Decision (10-14 days)
**Success metric:** 1st client booked by March 31, 2026
**Status:** Ready to implement with Karl agent + Tris approval workflow
