## Detailed Recommendations

### 1. ONLINE REVIEWS - Biggest Opportunity
**Problem:** 0 reviews = hesitation from prospects on $30K-$100K projects
**Solution:** 
- Send follow-up email/text after every project with direct links
- Aim for 20+ Google reviews in 6 months
- Even 10-15 quality reviews transforms perception
- Small incentive for honest feedback (not for positive reviews)

### 2. PORTFOLIO - Show Real Work
**Problem:** ~10 images, some AI-generated, no real project details
**Solution:**
- Photograph EVERY project (before/during/after)
- Create case study pages: scope, timeline, neighborhood, client goals
- 5-10 well-documented real projects > dozens of generic images

### 3. SEO & CONTENT MARKETING
**Problem:** Competitors have suburb-specific pages (Skokie, Evanston, etc.)
**Solution:**
- Create location landing pages for each suburb served
- Start blog: 2-4 posts/month (cost guides, permits, design trends)
- Builds organic traffic over time

### 4. TRUST SIGNALS
**Problem:** Minimal credibility markers vs competitors with awards/press/bios
**Solution:**
- Add About page with team photos + bios
- Display certifications/licenses prominently
- BBB profile
- Houzz listing → aim for Best of Service award
- Add press mentions / community involvement

### 5. DIRECTORY PRESENCE
**Problem:** Only on Yelp (0 reviews), missing Houzz/HomeAdvisor/Thumbtack/Angi/BBB
**Solution:**
- Claim/create profiles on all major platforms
- Upload best photos to each
- Keep NAP (name/address/phone) consistent
- Fast response time = ranking factor

### 6. WEBSITE ENHANCEMENTS
Add missing pages:
- **About/Team** - Photos + story (who they're hiring)
- **Process** - 6-step consultation-to-completion flow
- **Testimonials** - Real client feedback (linked to reviews)
- **FAQ** - Timelines, permits, pricing expectations
- **Videos** - Even phone walkthroughs add credibility
