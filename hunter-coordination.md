## Coordination Rules

**Before Tris calls:**
- Mark target as "Tris - Called" + date + outcome
- Don't call if Hunter already emailed (wait 3 days first)

**Before Hunter emails:**
- Check Lead Tracker to see if Tris already called
- If yes: reference the call in the email ("I know Tris reached out...")
- If no: start fresh with cold email

**Lead Tracker Columns:**
- Name / Company
- Phone
- Email
- Tris Status (Not Contacted / Called / Interested / No Response)
- Tris Date
- Hunter Status (Not Contacted / Email Sent / Opened / Replied / No Response)
- Hunter Date
