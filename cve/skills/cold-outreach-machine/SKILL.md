# Cold Outreach Machine

Turn a target market into booked meetings with personalized outreach at scale.

## Process

1. **Define the campaign:**
   - Your business/service
   - Target audience (who are you reaching?)
   - Platform (email, LinkedIn DM, Instagram DM, SMS, Facebook)
   - Goal (book call, get reply, drive to page)
   - Offer (free consultation, discount, demo, value-add)

2. **Research prospects:**
   - Use web search to find prospects in target market
   - Pull: name, business, role, recent activity, pain points
   - For local businesses: Google Maps search, Yelp, social profiles
   - For B2B: LinkedIn, company websites, recent news

3. **Write personalized messages:**

### First Touch Template Structure
```
Line 1: Personal hook (reference their work, post, or achievement)
Line 2: Bridge to the problem you solve
Line 3: Your value prop (specific, not generic)
Line 4: Soft CTA (question, not demand)
```

### Rules
- **No "I hope this finds you well"** — start with THEM, not you
- **Specific > generic** — reference their actual business/content
- **Short** — Email: 3-5 sentences. DM: 2-3 sentences. SMS: 1-2 sentences.
- **One CTA** — don't ask them to do 3 things
- **No attachments** on first touch
- **Sound human** — typos are ok, robot-speak is not

4. **Follow-up sequence (3 touches):**

```
Day 0: First touch (personalized intro)
Day 3: Follow-up 1 (add value — share tip, resource, or insight)
Day 7: Follow-up 2 (break-up email — "totally understand if now isn't right")
```

5. **Output format:**

For each prospect, generate:
```
## Prospect: [Name] — [Business]
Research: [What you found about them]
Platform: [Where to reach them]

Message 1 (Day 0):
[Full message ready to send]

Message 2 (Day 3):
[Follow-up message]

Message 3 (Day 7):
[Break-up message]
```

## Batch Mode
For campaigns of 10+ prospects, output as a table:
| Prospect | Business | Platform | Message 1 | Follow-up 1 | Follow-up 2 |

## Tracking
After sending, track:
- Sent / Opened / Replied / Booked
- Which messages got highest reply rates
- Iterate on winning templates
