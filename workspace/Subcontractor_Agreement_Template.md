# Subcontractor Agreement Template

**For: Valencia Construction Services**  
**Purpose:** Protect both Valencia and subcontractors with clear terms on scope, payment, liability, and insurance.  
**Status:** Template (customize for each subcontractor)

---

## HOW TO USE THIS TEMPLATE

This agreement protects you when hiring electrical, plumbing, HVAC, or other licensed subcontractors. Customize the bracketed sections for each project and subcontractor.

**When to use:**
- Any work requiring a license (electrical, plumbing, HVAC, gas)
- Specialized work (tile, flooring, painting contractors)
- Any subcontractor working on Valencia projects

**What to do:**
1. Fill in subcontractor info, scope, price, dates
2. Both parties sign and date
3. Keep one copy on file for the project
4. Provide copy to subcontractor

### 8. CHANGE ORDERS

**Any changes to scope must be documented in writing.**

If the client requests additional work, or unforeseen conditions require changes:
1. Subcontractor notifies Contractor immediately
2. Contractor and subcontractor agree on additional cost (if any)
3. Written change order is signed by both before work proceeds
4. No additional payment without written change order

**Example unforeseen conditions:**
- Hidden wall cavity requires additional framing
- Old wiring found, needs replacement
- Pipe location different than expected
- Structural issue requiring adjustment

### 12. CLIENT COMMUNICATION

**Communication rule:** Subcontractor does NOT communicate directly with client regarding:
- Price changes
- Timeline changes
- Warranty issues

**All communication goes through Contractor.** If client asks you something, say: "Let me check with the Contractor and get back to you."

This protects both parties and keeps project coordination clear.

### 13. LIABILITY & DISPUTES

**Liability:**
- Subcontractor is liable for damage caused by their work (property damage, injury due to negligence)
- Subcontractor's insurance should cover these claims
- Contractor is not responsible for subcontractor's accidents or negligence

**If damage occurs:**
1. Report immediately to Contractor
2. Insurance claim filed by subcontractor
3. Repair/resolution coordinated through insurance

**Disputes:**
- Minor disagreements resolved by conversation between Contractor and Subcontractor
- If unresolved, mediation attempted before legal action
- Either party can pursue legal remedy if necessary

---

### 14. TERMINATION

**Contractor may terminate this agreement if:**
- Subcontractor fails to provide proof of insurance
- Subcontractor's work is consistently below standard
- Subcontractor abandons the job (no-show for 3+ days)
- Subcontractor violates safety or code requirements
- Subcontractor is unlicensed or misrepresents qualifications

**Upon termination:**
- Subcontractor is paid only for completed, approved work
- Subcontractor must leave site immediately
- Contractor may hire replacement to complete work; cost difference is subcontractor's responsibility

## VARIATIONS FOR DIFFERENT TRADES

### Electrical Subcontractor (Additional)
- Must have current Illinois Electrician license
- Responsible for all electrical code compliance
- Must pull permits for major work (or confirm Contractor pulling)
- Must schedule city inspection

### Plumbing Subcontractor (Additional)
- Must have current Illinois Plumber/Plumbing Contractor license
- Responsible for all plumbing code compliance (Chicago has strict codes)
- Must schedule city inspection
- Responsible for rough-in and final plumbing per plans

### HVAC Subcontractor (Additional)
- Must have Illinois HVAC license (if required)
- Responsible for ductwork, equipment installation per code
- Must schedule city inspection
- Responsible for system testing and startup

### Tile/Flooring Subcontractor
- No state license typically required, but verify
- Responsible for substrate prep, proper installation per manufacturer specs
- Responsible for grout sealing if included
- Warranty on material installation (not material defects)

## RED FLAGS (Don't Sign If...)

- ❌ Subcontractor won't provide proof of insurance
- ❌ License is expired or suspended
- ❌ No physical address, only PO box
- ❌ Refuses to sign agreement
- ❌ Demands full payment upfront (major red flag)
- ❌ Won't provide references
- ❌ Vague about scope ("We'll figure it out")
- ❌ No response to communication attempts
