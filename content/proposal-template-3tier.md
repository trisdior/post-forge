# QUICK-FILL EXAMPLES

## Example: Kitchen Remodel

**Option 1 — Essential ($12,000-18,000):**
- Demo existing countertops and backsplash
- Install new laminate countertops
- Install subway tile backsplash
- Paint existing cabinets (2 coats)
- Replace cabinet hardware
- Install new kitchen faucet
- Install new LVP flooring

**Option 2 — Premium ($22,000-35,000):** ⭐
- Full cabinet replacement (semi-custom)
- Quartz countertops
- Designer tile backsplash
- New undermount sink + faucet
- Updated lighting (under-cabinet + pendant)
- New LVP or hardwood flooring
- Fresh paint

**Option 3 — Luxury ($40,000-60,000+):**
- Custom cabinets (soft-close, specialty finishes)
- Premium stone countertops (marble, quartzite)
- Custom tile work (full wall, designer patterns)
- Premium fixtures throughout
- Electrical updates (new circuits, smart switches)
- Plumbing relocation if needed
- Custom range hood
- Designer hardware

## Example: Bathroom Remodel

**Option 1 — Essential ($5,000-8,000):**
- New vanity + top (stock)
- New toilet
- New faucet + showerhead
- Paint walls
- Replace flooring (LVP or sheet vinyl)
- Caulk + grout refresh

**Option 2 — Premium ($12,000-20,000):** ⭐
- New tiled shower/tub surround
- New vanity (semi-custom) + stone top
- New toilet (elongated, comfort height)
- Tile floor
- New fixtures (rain shower, widespread faucet)
- Updated lighting + mirror
- Fresh paint
- Exhaust fan upgrade

**Option 3 — Luxury ($25,000-40,000+):**
- Full gut to studs
- Custom tile (large format, accent wall, niche)
- Frameless glass shower door
- Heated floors
- Custom vanity or double vanity
- Premium fixtures (matte black, brushed gold)
- Freestanding tub (if space allows)
- Custom lighting plan
