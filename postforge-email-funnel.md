## Implementation Notes

**Tools Needed:**
- Email service: Resend (already set up for Tris)
- Segment triggers: Based on user activity (posts generated, days since signup, tier)
- Personalization: User name, post count, industry preset data

**Metrics to Track:**
- Open rate (target: 40%+)
- Click-through rate (target: 15%+)
- Conversion rate (free → pro, target: 5%+)
- Churn prevention (24h cancellation recovery, target: 20% save rate)

**A/B Testing:**
- Subject line variants (urgency vs. curiosity)
- CTA button text ("Upgrade Now" vs. "See What's Possible")
- Timing (24h vs. 48h vs. 72h delays)
- Personalization vs. template simplicity

**Pro User Retention:**
- Monthly engagement reports (show real data)
- Feature suggestions based on usage
- Upgrade path to Business tier at 80% of monthly quota
